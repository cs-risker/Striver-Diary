# 四、列表

## 无序列表

语法：

```
*/- 内容
```

* 同级列表行一
* 同级列表行二
* 同级列表行三



* 多级列表第一级
  * 多级列表第二级（使用tab）
    * 多级列表第三级

## 有序列表

语法：

```
数字+.+空格
```

1. 有序列表第一行
2. 有序列表第二行
3. 有序列表第三行

## 任务列表

语法：

```
* [ ]
```

* [ ] 背单词
* [x] 吃饭

# 八、脚注

说明：对文本进行解释说明

语法：

```
[^文本]
[^文本]:解释说明<br>实现回车
```

c语言[^1]

[^1]:一种编程语言<br>回车
